This is a PHP extension with common DDD files.

## Install

Via Composer

```bash
$ composer require vova07/DDD
```

## Usage

```php
$skeleton = new vova07\Skeleton();
echo $skeleton->echoPhrase('Hello, World!');
```

## Testing

```bash
$ phpunit
```
